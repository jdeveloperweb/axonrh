Visão Geral da Arquitetura

O AxonRH é uma plataforma de gestão de recursos humanos distribuída, construída sobre uma arquitetura de microserviços escalável. O sistema utiliza Spring Boot para o backend, React para o frontend e uma infraestrutura robusta baseada em containers Docker. A comunicação é orquestrada via API Gateway e mensageria assíncrona.

Stack Tecnológico

Camada Tecnologias principais Detalhes
Frontend React 18, Vite, TypeScript Single Page Application (SPA), Design System Customizado.
Backend Java 21, Spring Boot 3.2 Arquitetura de Microserviços, Spring Cloud.
Banco de Dados PostgreSQL 15, MongoDB 7 Postgres para dados relacionais, Mongo para logs de conversa/IA.
Cache & Sessão Redis 7 Cache distribuído, armazenamento de sessões de usuário.
Mensageria RabbitMQ 3 Comunicação assíncrona entre serviços (Event-Driven).
Storage MinIO (S3 Compatible) Armazenamento de documentos, fotos e arquivos.
DevOps Docker, Docker Compose Containerização total de todos os serviços.
