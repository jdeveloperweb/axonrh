Manual Técnico Detalhado da Arquitetura de Microserviços

Diferenciais da Arquitetura

Microserviços Puros
Desacoplamento total permitindo deploy independente, tolerância a falhas e escalabilidade horizontal.
Event-Driven
Comunicação assíncrona via RabbitMQ garante alta performance e resiliência em picos de carga.
Segurança OAuth2
Autenticação centralizada e autorização granular via JWT em todos os endpoints da malha.
Alta Observabilidade
Monitoramento distribuído, logs centralizados e Health Checks ativos em todos os containers.

Catálogo de Microserviços

API Gateway
Infraestrutura • Porta 8080

O porteiro do sistema. Utiliza Spring Cloud Gateway para rotear todas as requisições externas para os serviços internos apropriados.

Responsabilidades
Roteamento, Rate Limiting, Filtro de Autenticação, CORS.
Config Service
Infraestrutura • Porta 8888

Servidor de configuração centralizada. Permite alterar propriedades do sistema sem precisar rebuildar os serviços.

Responsabilidades
Hospedagem de configuration (application.yml) para todos os perfis (dev, prod).
Auth Service
Segurança • Porta 8081

Gerencia identidades, usuários e permissões. Emite tokens JWT assinados que são validados pelos outros serviços.

Endpoints Principais
  • /login
  • /register
  • /validate
Dados
PostgreSQL (Tabelas: users, roles, permissions)
Employee Service
Core RH • Porta 8083

O coração dos dados pessoais. Gerencia o cadastro completo dos colaboradores, documentos e histórico contratual.

Funcionalidades
Admissão (Onboarding), Gestão de Documentos (S3), Atualização Cadastral.
Core Service
Negócio • Porta 8082

Gerencia a estrutura corporativa. Mantém a hierarquia de departamentos, cargos, centros de custo e unidades de negócio.

Dados
PostgreSQL (Tabelas: departments, jobs, cost_centers)
Timesheet Service
Operacional • Porta 8084

Controle de frequência. Registra batidas de ponto, calcula horas trabalhadas, extras e banco de horas.

Endpoints
  • /clock-in
  • /report/monthly
AI Assistant Service
Inovação • Porta 8088

Módulo inteligente integrado à OpenAI. Atua como assistente virtual para tirar dúvidas de RH e analisar sentimentos.

Tecnologia
MongoDB (Histórico de chats), Integração OpenAI GPT-4.
Vacation Service
Operacional • Porta 8085

Gestão de ausências. Calcula períodos aquisitivos, saldos de férias e processa fluxos de aprovação de solicitações.

Performance Service
Estratégico • Porta 8086

Gestão de talentos. Ciclos de avaliação de desempenho, PDI (Plano de Desenvolvimento Individual) e Feedbacks.

Learning Service
Estratégico • Porta 8087

LMS Corporativo. Gerencia catálogo de treinamentos, matrículas e progresso dos colaboradores em trilhas de ensino.

Notification Service
Suporte • Porta 8089

Hub de comunicação. Escuta filas do RabbitMQ para enviar emails transacionais e push notifications para usuários.

Integration Service
Integração • Porta 8090

Gateway de arquivos. Processa arquivos bancários (CNAB), layouts de folha de pagamento e outras integrações batch.
